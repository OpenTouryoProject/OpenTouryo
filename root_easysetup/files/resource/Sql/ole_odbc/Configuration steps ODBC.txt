・[管理ツール]から[データソース（ODBC）]を選択

・[ODBC データ ソース アドミニストレータ]ダイアログが表示される
　ので[システムDSN]タブを選択し、[追加]ボタンを押下する。

・[データソースの新規作成]ダイアログが表示されるので
　[Driver do Microsoft Access（*.mdb）]を選択し[完了]ボタンを押下する。

・[ODBC Microsoft Accessセットアップ]ダイアログが表示されるので
　[データソース名]欄に任意のIDを、[説明]欄に説明を入力する。

・データベースの[選択]ボタンを押下し、
　[データベースの選択]ダイアログで
　C:\root\files\resource\db の Northwind.mdb
　を選択し[OK]ボタンを押下する。

・[ODBC Microsoft Accessセットアップ]ダイアログで[OK]ボタンを押下する。
・[ODBC データ ソース アドミニストレータ]ダイアログで[OK]ボタンを押下する。

---

· Select the [(ODBC) data source] from [Administrative Tools].

· [ODBC Data Source Administrator] dialog is displayed 
  Select the [System DSN] tab, pressing the [Add] button so. 

· [Create New Data Source] dialog box is displayed 
  Pressing the [Finish] button to select the [Driver do Microsoft Access(*.mdb)]. 

· [ODBC Microsoft Access Setup] dialog box is displayed 
  Enter a description in the [Description] column, any ID in the [Data Source Name] field. 

• The user presses the database the [Select] button, 
  In the [Select Database] dialog 
  Northwind.mdb in C:\root\files\resource\db
  Pressing the [OK] button to select. 

· Pressing the [OK] button in the [ODBC Microsoft Access Setup] dialog. 
· Pressing the [OK] button in the [ODBC Data Source Administrator] dialog.
